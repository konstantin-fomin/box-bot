# Telegram-бот "Коробки"

## Стек
- Python 3, aiogram 3.x
- SQLite (файл data/boxes.db)
- qrcode (генерация QR-кодов)
- systemd для автозапуска
- Хранение фото — только file_id из Telegram, файлы не скачивать

## Функционал

### Создание коробки
Кнопка "📦 Новая коробка" → бот спрашивает комнату/категорию (кухня, спальня, детская и т.д.)
→ генерирует код вида ROOM-01 (автоинкремент номера по комнате)
→ просит прислать список вещей (текстом или голосом)
→ просит прислать фото (можно несколько, можно пропустить)

### Голосовые сообщения
Распознавание речи через облачный API (НЕ локальный Whisper — на VPS 951MB RAM не хватит)

### Карточка коробки (inline-кнопки)
📦 ROOM-01
Статус: 🏠 дома
Комната: Кухня
Вещи: кастрюля, тарелки, блендер
[фото если есть]

Кнопки:
[✏️ Добавить вещь] [🗑 Удалить коробку]
[📷 Добавить фото] [🔄 Сменить статус]

### Статусы
🏠 дома / 📦 склад / 🚚 в пути / ✅ распаковано
Смена статуса — через inline-кнопки (выбор из списка)

### Поиск
"где блендер?" → бот ищет по всем коробкам, показывает карточку(и), где есть совпадение
Поддержка нечёткого поиска (подстрока, без учёта регистра)
Автоподсказки при вводе (опционально, если легко реализуется)

### Список коробок
Кнопка "📋 Список коробок" → inline-кнопки по каждой коробке (код + комната)
Фильтры: по статусу, по комнате

### QR-коды
Генерация QR-кода с кодом коробки (текст ROOM-01) при создании
Отправка картинки QR пользователю для печати
Сканирование (просто через камеру Telegram/QR-сканер) → откроется чат с ботом с командой /box_ROOM-01 → бот показывает карточку

### Экспорт
Кнопка "выгрузить всё" → генерация PDF или таблицы со всеми коробками (код, комната, статус, список вещей)

### Общий доступ
Несколько Telegram-пользователей (whitelist по user_id в конфиге) видят и редактируют одну базу

### Reply-клавиатура (всегда внизу)
📦 Новая коробка | 🔍 Найти вещь
📋 Список коробок | ⚙️ Ещё

## Структура БД (SQLite)
boxes: id, code, room, status, created_at
items: id, box_id, name
photos: id, box_id, file_id
users: user_id, name (whitelist для доступа)

## Развёртывание
- polling (не webhook), без домена и SSL
- systemd unit для автозапуска и рестарта при сбое
